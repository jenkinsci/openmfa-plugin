# OpenMFA Plugin — Documentation

This directory holds documentation and supporting assets for the OpenMFA Jenkins plugin.

- **`init.groovy.d/`** — Jenkins init scripts (e.g. `users.groovy` for local/dev user setup). For local and development use only; do not commit real credentials.
- **`notes/`** — Technical notes, troubleshooting, and knowledge gathered during development (Jenkins/Stapler, TOTP, publishing, secrets, etc.). See `notes/README.md` for structure and how to contribute.
